SRAM Fingerprint

Project summary

When a Static Random-Access Memory (SRAM) chip is powered up it has a unique fingerprint. Each bit contained in every memory address will flip to either zero or one. These values can be used to identify the chip. This project will focus on finding ways to measure and change this fingerprint, using such values to identify chips, and investigating if the chips manufacturer can be identified (what we call the DNA of the chip).

Project background

SRAM and the SRAM fingerprint has many applications. Current computers use whats called a pseudo random number generator(PRNGs). These PRNGs are vulnerable to hacks, for many purposes the randomness is “good enough”. But when used for security can become a single point of failure. The SRAM fingerprint could potentially be used to produce true random numbers. Another application is identification. If a country or individual could get access to a laptop containing sensitive information that is encrypted, they could potentially swap out components to try and can access. If these parts could be identified by a physical unclonable function it could be a way to mitigate this. The last application relevant to our project is DNA. It could potentially be possible to identify the manufacturing process and manufacturer based off of these values. PRNG source - https://www.unitychain.io/blog/a-brief-history-of-random-insecurity/
